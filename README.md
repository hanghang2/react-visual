# react visual

## 项目目录

> index.js (项目入口文件)  
> RouterContainer.jsx (路由组件)  
> serviceWorker.js (后台运行的一个线程，可以用来处理离线缓存、消息推送、后台自动更新等任务)  
> setupTests.js (测试脚本)  
> api (请求)  
>> axios.js (请求处理)  
>> index.js (请求封装)  

> store (状态管理)  
>> index.js (入口)  
>> reducer.js (状态管理)  

> style (样式)  

> view (页面)  
>> ConfigPage (拖拽组件页面)  
>> Home (首页)  
>> Login (登陆页)  

> components
>> business (业务组件)  
>> container (容器组件)  
>> common (公共组件)
>>> ComponentAlert (组件选择弹框)  
>>> ComponentSetting (组件属性设置)  
>>> HomeList (首页视图list)
